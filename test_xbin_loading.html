XBin Font/Palette Loading Test

This page can be used to manually test XBin file loading in sequence.

Test Steps:

  1. Open the main application at http://localhost:8082
  2. Click the character brush tool (B key)
  3. Load examples/xbin/xz-neuromancer.xb
  4. Observe console and UI
  5. Load examples/xbin/xz-xero.xb
  6. Observe console and UI for any "PNG Font glyph not available" warnings
  7. Test loading the files in reverse order

Expected Behavior:

  • No "PNG Font glyph not available" warnings
  • Character brush panel should show the correct font after each load
  • Floating panels should show the correct palette colors
  • Main canvas should render correctly

Files to Test:

  • examples/xbin/xz-neuromancer.xb - Custom blue/gray palette
  • examples/xbin/xz-xero.xb - Custom color scheme
  • examples/xbin/xz-divinestylers.xb - Another custom palette
